This Thursday I’ve given a talk at Berlin R-Users Group. The topic was “Touring the tidyverse: tidy data + tidyr”. The idea is that this will become a series of talks where each consequent talk is going to present one (or couple) of packages from the tidyverse.

tidyr to me seems like a good choice for the first talk since concept of tidy data is so central to all of the packages. Therefore, understanding this concept makes it easier to work in tidyverse. The code for the talk and presentation itself are available at my github - https://github.com/romatik/touring_the_tidyverse.

All talks in the series:

  1. tidyr - https://www.mishabalyasin.com/2018/05/27/touring-the-tidyverse-tidyr/
  2. dplyr - https://www.mishabalyasin.com/2018/07/26/touring-the-tidyverse-dplyr/
  3. purrr - https://www.mishabalyasin.com/2018/10/24/touring-the-tidyverse-purrr/
  4. tidyeval - https://www.mishabalyasin.com/2018/11/30/touring-the-tidyverse-tidyeval/
